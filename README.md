# EasyDownloader - Chrome浏览器插件

## 功能描述
EasyDownloader是一个Chrome浏览器插件，可以通过右键点击下载页面上的资源，如图片、视频、表格数据等。

## 主要功能
1. 通过右键点击，可以出现一个EasyDownloader菜单
2. 点击EasyDownloader菜单后，会出现一个选择资源下载的对话框
3. 对话框中可选择的资源列表是当前页面中的图片、视频、表格等资源
4. 列表以表格形式出现，每行记录前面有一个复选框，表示是否选中当前资源
5. 表格中清晰显示资源名称、类型等信息
6. 当选择资源确定后，开始下载对应的资源，并保存到本地文件

## 安装方法
1. 在Chrome浏览器中打开扩展管理页面：chrome://extensions/
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本项目的根目录

## 注意事项
1. 图标文件目前为SVG格式，如需使用PNG格式，请使用工具将SVG转换为PNG
2. 某些网站可能有防止下载的保护措施，可能导致部分资源无法下载
3. 下载的文件将保存在浏览器默认的下载目录中

## 使用说明
1. 在任意网页上右键点击
2. 在弹出的上下文菜单中选择"EasyDownloader"
3. 在弹出的对话框中选择要下载的资源
4. 点击"下载所选"按钮开始下载
